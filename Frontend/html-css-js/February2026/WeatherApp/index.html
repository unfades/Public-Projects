🌨️Weather App🌥️

Get your weather today!

This is my mini project playing with Openmeteo's Weather API. It allows for anyone to use the API for under 10k requests every month. More requests? Then you have to use the payed version, which is still a pretty good deal. It offers different metrics in order to utilize all of the data that it provides. It even provides weather history as well within it. That's a pretty nifty feature. As far as the weather information goes, it is quite accurate.
